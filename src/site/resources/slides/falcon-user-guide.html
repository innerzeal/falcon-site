Apache Falcon - User Guide

Building Apache Falcon

  1. Clone the project

    git clone https://git-wip-us.apache.org/repos/asf/incubator-falcon.git

  2. Compile the project

    mvn -DskipTests clean package

  3. Optionally run the tests

    mvn test

Deploying Apache Falcon

  • Deploy Falcon webapp into Tomcat container

  • Start Apache ActiveMQ

  • Start Apache Tomcat

Validating Apache Falcon

  • Falcon is running at: http://localhost:15000/falcon
